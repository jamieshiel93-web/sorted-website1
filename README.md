# Sorted. Website

This package preserves your chosen `index.html` exactly.

Upload these files to the root of your GitHub repository:

- index.html
- README.md
- robots.txt
- sitemap.xml
- site.webmanifest
- favicon/site-icon.svg

Do not upload the ZIP file itself.

After uploading, your site should be available at:
https://jamieshiel93-web.github.io

If GitHub takes a few minutes to update, refresh after 2–5 minutes.
